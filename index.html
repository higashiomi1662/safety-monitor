🔵 Google ドライブ 連携設定

Google Apps Script（GAS）を使ってデータをドライブへ保存します。
設定は一度だけ。以降はボタン1つでバックアップできます。

STEP 1 — Google Apps Script を作成
  1. script.google.com を開く（Googleアカウントでログイン）
  2. 「新しいプロジェクト」を作成
  3. 下のコードをコピーしてエディタに貼り付ける
  4. 「デプロイ」→「新しいデプロイ」→ 種類：ウェブアプリ
    「アクセスできるユーザー：全員」に設定して「デプロイ」
  5. 表示された「ウェブアプリURL」をコピーして下に貼り付ける
GASコード（コピーして使用）：
function doPost(e) { const data = JSON.parse(e.postData.contents); const folder = DriveApp.getRootFolder(); const fname = 'safety_backup_' + data.timestamp + '.json'; const blob = Utilities.newBlob( JSON.stringify(data.payload, null, 2), 'application/json', fname ); folder.createFile(blob); return ContentService .createTextOutput(JSON.stringify({ok:true, file:fname})) .setMimeType(ContentService.MimeType.JSON); } function doGet(e) { const files = DriveApp.getRootFolder().getFilesByName('safety_backup_latest.json'); if (files.hasNext()) { const content = files.next().getBlob().getDataAsString(); return ContentService .createTextOutput(content) .setMimeType(ContentService.MimeType.JSON); } return ContentService.createTextOutput('{}').setMimeType(ContentService.MimeType.JSON); }
👆 クリックでクリップボードにコピー
